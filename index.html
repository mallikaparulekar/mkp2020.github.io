https://github.com/mkp2020/debate_counter_arguments : automatically detecting counter arguments to evidence given in policy debate tournaments.

When evidence is given by a team in the context of a larger piece of text, this code automatically scans it and tries to find counter arguments to the evidence in the same piece of text.
Here this maximum limit may be 300 000 000 000 and more.


https://github.com/mkp2020/gene_sequence_clustering : Python code to cluster genetic data.

Python code to cluster genetic data. Implements clustering of gene sequences and determines probable promoter regions in each cluster. Uses Dirichlet distribution to generate different kinds of test data by varying alpha values.
